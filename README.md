# 🔥 Trending Searches in the US - Data Analysis Project

## 📊 Project Overview

This project analyzes trending search data in the United States using Python.  
It explores how search interest changes over time and identifies the most popular search topics.

The goal is to practice **data analysis, visualization, and pattern detection** using real-world data.

---

## 📁 Dataset

The dataset used in this project:
- `trending_searches_in_us.csv`

It contains information about:
- Search terms
- Frequency / popularity
- Time-based trends

---

## 🛠️ Tools & Libraries Used

- Python 🐍
- Pandas 📊 (data handling)
- NumPy 🔢 (calculations)
- Matplotlib 📈 (visualizations)

---

## 📊 Key Analysis Performed

This project includes:

- Loading and cleaning the dataset
- Finding most trending search terms
- Analyzing frequency of searches
- Visualizing top searches using bar charts
- Identifying patterns in search behavior
